# Activity Diagram

## Project: Smart Cab Security Platform

### Objective

The Activity Diagram illustrates the workflow of the Smart Cab Security Platform from booking a cab to completing the trip and handling emergencies.

---

## Workflow

1. Passenger opens the application.
2. Passenger logs in.
3. Passenger enters pickup and destination.
4. System searches for available drivers.
5. Driver accepts the booking.
6. System confirms the booking.
7. Passenger starts the trip.
8. AI Safety Module monitors the trip.
9. Emergency detected?
   - If **No**, continue the trip.
   - If **Yes**, send SOS alert and notify Admin/Emergency Contacts.
10. Passenger reaches destination.
11. Payment is completed.
12. Trip ends successfully.
