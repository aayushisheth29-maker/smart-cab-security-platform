# System Architecture

## Overview

The Smart Cab Security Platform consists of five major components:

1. Passenger Mobile Application
2. Driver Mobile Application
3. Backend Server
4. Database
5. AI Safety Monitoring Module

## Architecture Flow

Passenger App
↓
Backend Server
↓
Database

Driver App
↓
Backend Server
↓
Database

AI Monitoring Module
↓
Backend Server
↓
Emergency Alert System

## Components

### Passenger Application
- User Registration
- Ride Booking
- SOS Button
- Live Location Sharing

### Driver Application
- Driver Login
- Ride Management
- Navigation Support

### Backend Server
- Authentication
- Ride Processing
- Alert Management
- API Services

### Database
- User Data
- Driver Data
- Ride History
- Incident Reports

### AI Safety Module
- Risk Detection
- Route Monitoring
- Emergency Analysis

### Emergency Alert System
- Notify Trusted Contacts
- Notify Taxi Company
- Share Live Location
