Objective

The State Diagram represents the different states of a cab ride from booking to completion. It also shows how the system responds during an emergency situation.

State Flow
Passenger opens the Smart Cab application.
Passenger requests a cab.
System searches for available drivers.
Driver accepts the booking.
Driver arrives at the pickup location.
Passenger boards the cab.
Ride starts.
During the ride, the passenger may press the SOS button if an emergency occurs.
The system sends the emergency alert to the Admin.
Admin takes necessary action.
After the issue is resolved, the ride continues.
Passenger reaches the destination.
Payment is completed.
Ride ends successfully.
