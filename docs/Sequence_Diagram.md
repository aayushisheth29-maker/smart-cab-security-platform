# Sequence Diagram

## Smart Cab Security Platform

The sequence diagram illustrates the interaction between the Passenger, System, Driver, and Emergency Alert Module during a cab booking process.

### Process Flow

1. Passenger requests a cab.
2. System validates the request.
3. System searches for available drivers.
4. Driver accepts the ride request.
5. System confirms booking.
6. Passenger starts trip.
7. During emergency, passenger sends alert.
8. Emergency Alert Module notifies Admin.
9. Admin takes necessary action.
